Module DisplayText

DisplayText.new(font, size, parttern, text)

Creates a new object of display text.

Parameters

  • font : (Font) The font of the display text.

  • size : (optional) Sets the current font matrix to a scale by a factor of size.

  • parttern : (optional) The parttern of the display text.

  • text : (optional) The content of display text that will be placed on the screen.
