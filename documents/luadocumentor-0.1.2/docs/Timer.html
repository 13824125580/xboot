Module Timer

The 'Timer' class is used to execute a code at specified intervals.

Type Timer

Timer:isRunning()

Returns the current running status of timer.
Timer.new(delay, iteration, listener)

Creates a new 'Timer' object with the specified delay and iteration.
Timer:pause()

Pauses the timer that was running.
Timer:resume()

Resumes the timer that was paused.

Type Timer

Field(s)

Timer:isRunning()

Returns the current running status of timer.

Return value

'true' if the timer is running, 'false' otherwise.

Timer.new(delay, iteration, listener)

Creates a new 'Timer' object with the specified delay and iteration.

Parameters

  • delay : (number) The delay in seconds

  • iteration : (number) The number of times listener is to be invoked. pass 0 if you want it to loop forever.

  • listener : (function) The listener to invoke after the delay.

Return value

New 'Timer' object.

Timer:pause()

Pauses the timer that was running.

Timer:resume()

Resumes the timer that was paused.
