Module TimerManager

The 'TimerManager' class is used to manager timer.

Type TimerManager

TimerManager:addTimer(timer)

Registers a timer
TimerManager:hasTimer(timer)

Checks if the 'TimerManager' object has a timer.
TimerManager.new()

Creates a new 'TimerManager' object.
TimerManager:removeTimer(timer)

Removes a timer from the 'TimerManager'.
TimerManager:schedule(dt)

Schedule timers according to time interval.

Type TimerManager

Field(s)

TimerManager:addTimer(timer)

Registers a timer

Parameter

  • timer : (Timer) The timer will be registered.

Return value

A value of 'true' or 'false'.

TimerManager:hasTimer(timer)

Checks if the 'TimerManager' object has a timer.

Parameter

  • timer : (Timer) The timer was registered.

Return value

A value of 'true' if a timer is registered; 'false' otherwise.

TimerManager.new()

Creates a new 'TimerManager' object.

Return value

New 'TimerManager' object.

TimerManager:removeTimer(timer)

Removes a timer from the 'TimerManager'.

Parameter

  • timer : (Timer) The timer will be removed.

Return value

A value of 'true' or 'false'.

TimerManager:schedule(dt)

Schedule timers according to time interval.

Parameter

  • dt : (number) The time delta in seconds.
